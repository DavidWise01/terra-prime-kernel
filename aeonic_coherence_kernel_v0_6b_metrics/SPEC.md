# Aeonic Coherence Kernel v0.6b — Metric Recalibration

v0.6 created true three-body behavior but the old stability metric saturated at 1.0.

v0.6b fixes that by splitting the overloaded metric into three measures.

## New Metrics

### Structural Stability

Can the system hold together?

Uses:
- identity drift
- bridge tension
- rigidity
- cohesion

Target:

```text
0.82–0.90
```

### Vitality

Is the system still alive, moving, and generating?

Uses:
- dissent
- emergence
- Mercury motion
- alliance shift

Target:

```text
0.65–0.90
```

### Viability

Can the system remain coherent while alive?

```text
viability = structural_stability × vitality-adjusted survival factor
```

## Three-Body Meaning

Light wants explicit order.
Shadow wants unresolved depth.
Mercury wants movement and exchange.

The old metric called reconciliation “perfect stability.”
The new metrics distinguish:

```text
holding together != being alive
```

## Goal

No more false 1.0 stability saturation.
