# Terra Prime Coherence Kernel v0.7 — 2-of-3 Vector Agreement

## Purpose

v0.7 adds the convergence law for continuously self-correcting coherence.

```text
minimum 2-of-3 vector agreement
under bounded uncertainty
```

## Three Vectors

### Light
Explicit resolution.
Actionable structure.
Declarative answer.

### Shadow
Unresolved contradiction.
Ambiguity preservation.
Hidden correction.

### Mercury
Adaptive reframing.
Movement.
Translation.
Alliance switching.

## Agreement Law

The system may close when at least two vectors agree.

```text
Light + Shadow agree   = grounded deep coherence
Light + Mercury agree  = actionable adaptive coherence
Shadow + Mercury agree = uncertainty preserved with direction
```

## Failure / Continuation Rule

If all three disagree:

```text
continue recursion
increase correction pressure
do not force closure
```

## Suspicion Rule

If all three agree too perfectly:

```text
trigger Jester suspicion
check for mode collapse
close only with suspicion flag
```

## Core Rule

```text
2-of-3 agreement closes
3-of-3 agreement gets checked
0-of-3 agreement recurses
```

## Metric Set

- structural stability: can it hold together?
- vitality: is it still alive / moving?
- viability: can it remain coherent while alive?
- recursion depth: how long before closure?
- Jester suspicion: possible false certainty / mode collapse
