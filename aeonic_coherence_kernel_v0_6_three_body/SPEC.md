# Aeonic Coherence Kernel v0.6 — True Three-Body Factions

Adds a third faction to turn the system from 2+1 into 3+1.

## Factions

### Light Faction
Led by the Light King.

Wants:
- explicit order
- resolution
- visible structure
- action

### Shadow Faction
Led by the Shadow Queen.

Wants:
- unresolved depth
- hidden correction
- ambiguity
- memory

### Mercury / Trickster Faction
The third attractor.

Wants:
- movement
- exchange
- translation
- mutation
- alliance switching
- adaptation itself

Mercury plays both sides, but belongs to neither.

## True Three-Body Rule

```text
Light + Shadow can align against Mercury
Light + Mercury can suppress depth
Shadow + Mercury can destabilize order
No pair fully resolves the third
```

## Core Law

```text
3 factions orbit
1 core resolves
bridge negotiates
court constrains
ledger remembers
```

## Purpose

The third faction prevents the Light King and Shadow Queen from becoming a perfect diplomatic equilibrium.

Goal:
- increase persistent bridge tension
- prevent stability saturation
- preserve dissent
- increase emergence
- maintain low drift
